# Notion Workspace

## Recommended top-level pages

- Home
- Vision
- MVP
- Backlog
- Research
- Avatars
- Tech Decisions

## Backlog database fields

- Title
- Status
- Priority
- Type
- Owner
- Target milestone
- Source link
- Notes

## Avatars database fields

- Name
- Slug
- Role
- Tone
- Core goal
- Memory summary
- Model hint
- Status

## Research database fields

- Title
- Topic
- Source type
- Link
- Summary
- Relevance
- Action needed

## Tech Decisions database fields

- Title
- Status
- Date
- Context
- Decision
- Consequences

## Setup checklist

1. Create a workspace named `avatar_ai`
2. Create the top-level pages above
3. Import or copy templates from the `notion/` folder
4. Add your Notion integration or MCP connection to the relevant pages
5. Share the `Backlog`, `Research`, and `Avatars` pages with that connection

## What Notion should own

- Product vision
- MVP scope
- Research notes
- Avatar design notes
- Decision logs

## What Notion should not own

- Source code
- Pull requests
- Release tags
- Local runtime config
