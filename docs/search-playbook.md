# Search Playbook

## Goal

Find reusable pieces without cloning random complexity into the MVP.

## Search themes

- Local LLM chat adapters
- Persona and memory formatting
- FastAPI chat APIs
- TTS integration
- Lightweight frontend chat UIs

## Good search phrases

- `ollama openai compatible chat fastapi`
- `llm persona memory yaml`
- `fastapi chat completions proxy`
- `local tts python streaming`
- `playwright scrape documentation`

## Where to search

- GitHub code search for implementation details
- GitHub repo search for project structure ideas
- PyPI for Python packages
- npm for frontend packages

## Scraping guidance

- Use Playwright when pages are dynamic or require login
- Use Scrapy when crawling many static pages with repeatable extraction rules
- Do not add scraping to the MVP unless there is a clear source and purpose
