# Vision

## Product idea

`avatar_ai` is a Telegram-first mini app for generating face-based image renders from a feed of ready-made templates.

## Target outcome

A user sees top templates in a Telegram channel or bot message, opens the mini app directly on a selected template, uploads a face once, starts a render from a saved face or a temporary override, waits in a free queue, and comes back to a simple result screen.

## Product principles

- Telegram-first user flow before public web
- Channel and bot posts create demand, mini app converts that demand inside the catalog
- Local-first development stack before cloud complexity
- Save the face once, then keep the template feed fast and obvious
- Queue-ready product shape before billing integrations
- Static, human-editable template definitions before admin tooling
- Small working architecture before real face-swap infrastructure

## Primary user

A solo builder or small team testing a viral avatar/face-template product inside Telegram before investing in production GPU infrastructure and payments.

## Non-goals for the first version

- Shorts or video generation
- Real face-swap backend integration
- Premium billing or payments
- Public consumer web app outside Telegram
- Persona chat as part of the main product UX
- Research or tooling dashboards in the user-facing flow
